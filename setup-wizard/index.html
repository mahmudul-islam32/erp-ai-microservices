Terminal Wizard Installation

macOS
Windows
Linux
1

Open Terminal

Press Cmd + Space, type "Terminal", and press Enter

2

Navigate to ERP Folder

In Terminal, run:

cd /path/to/erp-ai-microservices
Replace /path/to/ with your actual folder location
3

Run the Setup Wizard

Copy and paste this command:

./setup.sh
4

Follow the Wizard

Answer a few questions about your preferences. The wizard will:

  • Check system requirements ✓
  • Ask for app name and settings
  • Create your admin account
  • Install everything automatically
  • Open your ERP in the browser
That's it! Your ERP will be ready in 5 minutes.

Automated Installation

Fastest installation with zero interaction. Uses smart defaults.

1

Open Terminal

Navigate to the erp-ai-microservices folder

2

Run This Command

./install.sh
3

Wait for Completion

The script will automatically:

  • Check system requirements
  • Generate secure passwords
  • Start all services
  • Create admin account
  • Load demo data

Manual Installation

For advanced users who want full control.

1

Create Configuration

cp .env.example .env

Then edit .env with your settings

2

Start Docker Containers

docker-compose up -d
3

Access Your ERP

Open browser to: http://localhost:5173

Login with default credentials (see documentation)
