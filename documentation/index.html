ðŸš€ Installation Guide

Important: Make sure Docker and Docker Compose are installed before proceeding.
1

Download & Extract

Extract the downloaded ZIP file to your desired location.

unzip erp-system.zip
cd erp-system

Extract the downloaded package

2

Configure Environment

Copy the environment template and configure your settings.

cp .env.example .env
nano .env  # Edit configuration
Tip: The system works with default values. You can configure later.
3

Start the Application

Launch all services with Docker Compose.

docker-compose up -d

Services starting with Docker Compose

4

Access Your ERP

Open your browser and navigate to the application.

Frontend: http://localhost:5173
Default Email: admin@example.com
Default Password: admin123

Login page - Use default credentials

Success! Your ERP system is now running. Change the default password immediately!
